# Tictac

An un-winnable game of Tic Tac Toe. The AI will always win or tie. Sorry :(

## Requirements
- ruby ~> 2.1.2
- bundler ~> 1.7
- rake ~> 10.0

## Installation

Clone the source and cd into the project directory.

Then execute:

    $ bundle install

Then build and install the gem:

    $ rake install

## Usage

The gem should install the 'tictac' binary.

Just run:

	$ tictac
